# My First Project

Welcome! This is your starter coding project. It's a simple interactive web page built with three core web technologies: **HTML**, **CSS**, and **JavaScript**.

---

## What's in this project?

| File | What it does |
|------|-------------|
| `index.html` | The structure of the page (like the skeleton of a house) |
| `style.css` | The appearance of the page (like paint, furniture, and decor) |
| `script.js` | The behavior of the page (like the lights, doors, and appliances) |
| `.gitignore` | Tells Git which files to ignore |

---

## How to run it

No installation needed! Just:

1. Find the `index.html` file on your computer
2. Double-click it to open it in your web browser
3. You'll see a to-do list app you can use right away

---

## How to make it your own

Here are some easy things to try:

**Change the title:**
Open `index.html` and find this line:
```html
<h1>Hello, World! 👋</h1>
```
Replace "Hello, World!" with your name or anything you like.

**Change the colors:**
Open `style.css` and find `color: #2d6a4f;` — that green color. Try changing it to:
- `red` or `blue` (plain English works too!)
- `#ff6b6b` for a coral pink
- `#4a90d9` for a sky blue

**Add more text:**
Open `index.html` and add a new paragraph anywhere inside `<div class="container">`:
```html
<p>This is my new paragraph!</p>
```

---

## Key concepts you just saw

- **HTML tags** like `<h1>`, `<p>`, `<button>` define what content appears
- **CSS rules** like `color:`, `font-size:`, `background-color:` control how it looks
- **JavaScript functions** like `addTodo()` make things happen when you interact
